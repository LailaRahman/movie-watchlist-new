## Reusable Badge Component (React)

### Description

This project demonstrates how to create and reuse a simple React component using props.

A reusable Badge component is created and rendered multiple times with different values to show reusability.

### Component Details

**Badge Component Props:**

* `label` – text label
* `value` – text or number value

Example usage:
<Badge label="Course" value="React" />

### Features

* Functional components only
* Reusable component with props
* Clean and readable JSX
* Built with React (Vite)
